# 労基署対策（立入検査の準備と当日対応）
立入検査は「勝ち負け」ではなく、**日頃の運用を“説明できる形”に整えるイベント**です。  
このページは一般的な実務整理です。最終判断は最新の法令・通達・所轄の運用で要確認。

## まずここだけ（60秒版）
- **窓口を一本化**：案内役／説明役／記録役を決める  
- **口頭より記録**：「やってます」より「記録がここにあります」  
- **その場で結論を急がない**：「確認して回答します」でOK  
- **期限管理が勝負**：指摘→対策→証拠→完了を1枚の表で回す

<a id="flow"></a>
## 立入検査の種類と流れ
一般的には、定期的なもの・申告/相談を起点にしたもの・災害/事故を起点にしたもの等があります（詳細は所轄運用で要確認）。

### 当日（入口）
- 来訪者の所属・氏名・用件を確認し、社内の対応責任者に連絡
- 対応開始時刻と対応者を記録（メモでOK）
- “全部出す”より、**論点に対して最短の資料**を提示

### 当日（現場確認）
- 指摘事項はその場でメモ（場所／対象／何を見たか）
- 可能なら写真（社内ルールの範囲で）
- わからない点は持ち帰り（「後ほど回答します」）

### 後日（是正・報告）
- 指摘を分類（記録不足／設備不備／教育不足／手順不足／体制不足）
- 応急対応＋恒久対策＋期限＋証拠を揃える
- 提出物はコピー/PDF保管し、最新版の管理を徹底

<a id="prep"></a>
## 事前準備チェックリスト（書類・運用）
※業種・規模・取扱いにより変わります。ここでは“抜けやすい順”のたたき台。

### 体制・運用
- [ ] 安全衛生管理体制（選任・委員会等）が説明できる
- [ ] 社内ルール（手順書・作業標準）が現場と一致している
- [ ] 年間計画／点検計画（簡単でOK、更新日が新しい）

### 記録（最重要）
- [ ] 教育記録（日時・対象・内容が1枚で追える）
- [ ] 点検記録（局排・換気・保護具など。補修の履歴があると強い）
- [ ] 健康診断・事後措置（未受診フォロー含む）
- [ ] リスクアセスメント（見直し日・優先順位が説明できる）

### 化学物質・有害要因（該当する場合）
- [ ] SDS（最新版）と現場の品名・ラベルが一致
- [ ] 保護具の選定根拠（何を基準にそのマスク/手袋か）
- [ ] 測定（作業環境測定／個人ばく露等）：計画→実施→評価→改善の流れが追える

<a id="onsite"></a>
## 当日対応の型（言い方・記録・提出）
### 使える言い方
- 「事実関係を確認して、記録で回答します」
- 「現時点の運用はこうです。根拠資料はこちらです」
- 「不明点は持ち帰って、○日までに回答します」

### 避けたい言い方
- 「たぶん大丈夫です」「昔からこうです」「問題になったことないです」

### 提出・提示のコツ（一般論）
- 原本の扱いは所轄運用で要確認。基本はコピー/PDFで社内保管を残す
- 論点に対して最短の資料を出す（“盛りすぎ”は逆に迷子になる）

<a id="correct"></a>
## 是正指導の進め方（期限管理）
最小フロー（これだけで回る）
1. 指摘事項を1行化（何が×か）
2. 分類（記録/設備/教育/手順/体制）
3. 応急対応（止血）→恒久対策（再発防止）
4. 期限（提出/完了/確認）を置く
5. 証拠（写真・記録・改訂履歴）を揃える
6. 1枚の表で全員が同じ進捗を見る

<a id="themes"></a>
## よくある指摘テーマ（安全衛生）
※一般的な観点。あなたの事業場での優先度は要確認。

<details>
<summary>1) 記録の整合（やっているのに“証拠がない”）</summary>

- 教育を実施した記録がない/薄い  
- 点検はしているが、是正の履歴が残っていない  
- 手順書はあるが、現場が違う運用になっている  

</details>

<details>
<summary>2) 設備の点検・不備対応（局排・換気など）</summary>

- 点検記録の欠落、頻度の不明確  
- 不具合が出た時の対応（補修・一時措置）の記録がない  

</details>

<details>
<summary>3) 化学物質管理（SDS・表示・保護具）</summary>

- SDSが古い/現場で参照できない  
- 容器表示が現場で徹底されていない  
- 保護具の選定根拠が説明できない  

</details>

## 分かること
- 立入検査の流れと、準備すべき資料の全体像。
- 当日の対応で避けたい言い方・記録の残し方。
- 是正指導の進め方と期限管理の型。

## 対象読者
- 労基署対応を初めて経験する現場担当者。
- 監督署対応の体制を整えたい管理者。
- 是正指導の記録整理を進めたい方。

## 使い方
1. 「まずここだけ（60秒版）」で全体像を掴みます。
2. 事前準備チェックリストを自社の状況に合わせて確認します。
3. 当日対応の型とテンプレを印刷・共有して運用します。

## FAQ
### Q1. 立入検査で必ず出さなければいけない書類は？
**A.** 所轄の運用で異なるため、基本の記録を準備しつつ要確認です。

### Q2. 指摘事項への回答はその場で決めるべき？
**A.** 即答より「確認して回答します」を基本にし、記録で対応します。

### Q3. 是正指導の期限管理が難しいです。
**A.** 指摘事項を1行化し、期限と証拠を1枚の表で管理すると回しやすいです。
