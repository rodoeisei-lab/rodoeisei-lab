## 1. 測定の目的

- 現場のリスクを見える化し、優先すべき改善を決める。
- 「測ること」よりも「どう改善に使うか」を先に決める。

## 2. 基本フロー

1. **計画**：対象物質・工程・作業時間を決める
2. **実施**：A測定/B測定など必要な方法を選ぶ
3. **評価**：管理区分を判定し、課題を明確化
4. **改善**：換気・工程・作業方法の見直し
5. **再評価**：改善後に再測定して有効性を確認

## 3. つまずきやすいポイント

- サンプリング地点が固定化され、実態を拾えていない。
- 高い数値の原因特定より先に対策を打ってしまう。
- 結果が報告書で止まり、現場改善までつながらない。

## 4. 現場で使う最小チェック

- 対象工程は最新の運転条件を反映しているか。
- 換気設備の稼働状態と測定タイミングが合っているか。
- 評価結果に対して、担当者と期限が設定されているか。

## 5. 次アクション

- 直近の測定結果を1つ選び、改善アクションを1件だけ決める。
- 改善期限と再評価日を同時に決め、更新ログに登録する。
