# 参加・投稿ガイド

全国の労働衛生担当者が、安心して知見を共有できる場にするためのガイドです。

## 参加目的

- 現場で起きる課題を、実務目線で持ち寄って解決のヒントを増やす。
- 法令・通達・一次情報に基づいた解釈を共有し、判断の質を高める。
- 匿名相談を起点に、他の担当者にも役立つ記事として知見を循環させる。

## 公開の場と匿名相談の使い分け

- **公開で相談・知見共有したいとき**: GitHub Issues を利用（自己紹介・公開Q&A）
- **匿名で個別に相談したいとき**: 既存フォームを利用

## 参加・相談の3ステップ

1. **ステップ1：目的・ルールを確認**
   - 本ページの手順と注意事項を確認してください。
2. **ステップ2：自己紹介を投稿（Issue Form）**
   - まずは自己紹介Issueを投稿して参加完了にします。
3. **ステップ3：質問・知見を投稿／共有**
   - 公開Q&A Issueへの投稿、または匿名フォーム投稿を選んでください。

## 投稿ルール（要点）

- 個人や企業が特定できる情報は記載しないでください。
- 緊急対応が必要な案件は、公的機関・専門機関への相談を優先してください。
- 投稿内容は、表現調整や再構成のうえで紹介する場合があります。

## 反映までの流れ（匿名フォーム）

1. フォーム受信
2. 内容確認（重複・公開可否の判断）
3. 回答方針の整理
4. 必要に応じて記事化・公開

## よくある質問

### 匿名投稿ってどう扱われるの？
投稿時点で個人情報を含めない運用を前提とし、公開時も特定につながる記述は避けます。

### 投稿はいつ反映される？
内容確認の順に対応します。テーマの重複や調査の要否によって反映時期が変わります。
