# seo-basics（SEO最低限整備スキル）

## 目的
- title / description / H1 / 見出し階層を整える

## 入力
- 各ページの見出し
- 本文

## 実行手順
1. ページ目的と検索意図を1ページずつ整理する
2. title（32文字前後）とdescription（80〜120文字）を作成する
3. H1の一意性を確認し、H2/H3の階層崩れを修正する
4. キーワードの過剰反復や抽象見出しを除去する
5. ページ別にNG例を明示する

## 出力
1. ページ別メタ案（title / description）
2. 見出し修正案（H1-H3）
3. NG例指摘

## ルール
- titleはページごとに重複禁止
- 見出しは内容を具体的に要約する
