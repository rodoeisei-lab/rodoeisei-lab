# accessibility-check（読みやすさ/アクセシビリティ点検スキル）

## 目的
- 文字サイズ・コントラスト・見出し構造・alt等をチェック

## 入力
- HTML
- CSS

## 実行手順
1. 見出し構造（H1-H3）とランドマークの欠落を確認
2. 本文サイズ・行間・余白の可読性を確認
3. 文字色と背景色のコントラストを確認
4. 画像alt、リンク文言、フォームラベルの不足を確認
5. 変更コストが低い順に修正提案を並べる

## 出力
1. 修正チェックリスト
2. 推奨変更（最小差分）

## ルール
- デザイン全面改修ではなく、最小差分で改善する
- 指摘には対象箇所と修正例をセットで示す
