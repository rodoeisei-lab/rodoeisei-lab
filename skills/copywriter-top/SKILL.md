# copywriter-top（トップ文言最適化スキル）

## 目的
- 「何のサイトか」を3秒で伝える見出し＆CTA作成

## 入力
- ターゲット
- 提供価値
- 主導線（例：人気記事 / 道具棚 / 問い合わせ）

## 実行手順
1. ターゲットの課題を1行で言語化する
2. 提供価値を「誰に・何を・どう変えるか」で再定義する
3. ヒーロー見出しを方向性違いで10案作る
4. 見出しに対応するサブコピーを作る
5. 主導線に直結するCTA文言を複数案作る

## 出力
1. ヒーロー見出し案10個
2. サブコピー
3. CTA文言

## ルール
- 曖昧語（すごい/便利）より具体便益を優先
- 1文を短くし、視認性を最優先する
