# content-planner（記事企画・編集カレンダースキル）

## 目的
- サイトの目的に沿って記事テーマを量産可能にする

## 入力
- ターゲット
- 既存記事
- 伸ばしたいカテゴリ

## 実行手順
1. 既存記事の不足トピックと重複を整理
2. 検索意図別（入門/比較/実践/事例）に企画を展開
3. 30本のテーマ案をカテゴリ配分つきで作る
4. 各テーマに構成テンプレ（見出し骨子）を付与
5. 公開順と内部リンク先を指定する

## 出力
1. ネタ30個
2. 構成テンプレ
3. 内部リンク先まで指定した編集計画

## ルール
- 単発記事ではなくシリーズ性を優先
- 既存資産へ送客できる内部リンクを必ず入れる
