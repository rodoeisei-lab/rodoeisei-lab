# performance-optimizer（速度改善スキル）

## 目的
- 画像最適化、不要JS削減、CLS防止など体感速度UP

## 入力
- 画像一覧
- HTML / CSS / JS

## 実行手順
1. 重量要因を分解（画像/JS/CSS/フォント/外部タグ）
2. 即効性の高い軽量化施策を抽出
3. CLS/LCP/INPに関わる箇所を優先順位付け
4. 実装手順と検証方法を明確化

## 出力
1. 具体的な軽量化手順
2. 変更優先度
3. 確認方法

## ルール
- まず「効果大・工数小」から着手する
- 変更ごとに測定指標（LCP/CLSなど）を紐づける
